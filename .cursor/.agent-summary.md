## 🔄 How It Works

### Deletion Order (Reverse Dependencies)
```
Properties → Societies → Localities → BHK Types → Property Types → Cities → Categories → Listing Types
```

### Seeding Order (Follow Dependencies)
```
Listing Types → Categories → Property Types → BHK Types → Cities → Localities → Societies
```
